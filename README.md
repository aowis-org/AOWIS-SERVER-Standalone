# AOWIS-SERVER-Standalone

Meta-Repository to compile a standalone version of the **AOWIS Controller**.

The baked-in server components are:

- AOWIS-SERVER-GUI (the *Controller*)
- AOWIS-SERVER-MAP (the *Caching Map Tile Server*)

## Build

Make sure you have also the Git Submodules checked out by running either

* `git_submodule_init.sh` or
* `git_submodule_update.sh`

After that, you can compile by running

- `compile_linux.sh`
